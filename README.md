# Andvaranaut
![](art/screenshots/2018-02-17-143059_600x300_scrot.png)

The open source dungeon crawler

## Dependencies
    SDL2-ttf-devel
    SDL2-devel

## Building
    make -C src; ./andvaranaut

## Recommended System Requirements
    RAM: 256 MB
    CPU: 4 cores or more
    GPU: Just about anything will do
