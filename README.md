# Andvaranaut

The open source dungeon crawler

![](art/screenshots/2018-09-30-083300_920x550_scrot.png)

## Dependencies
    SDL2-ttf-devel
    SDL2-devel

## Building
    make -C src; ./andvaranaut

## Recommended System Requirements
    RAM: 256 MB
    CPU: 4 cores or more
    GPU: Just about anything will do
